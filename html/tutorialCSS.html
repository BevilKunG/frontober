CSS

CSS - Cascading Style Sheets เพิ่มความสวยความงามให้แก่เว็บไซต์

selector {
key: value;
}

Selector

By Element - element

เหมาะกับการ set style เป็นมาตรฐานของเว็บ
Example
<p> This element goes red </p>
p {
color: red;
}

This element goes red

By Id - #id

เหมาะกับการ query เพื่อใช้กับ javascript ไม่ค่อยเหมาะกับการ set style
Example
<p> Text is white, Background is Blue </p>
#id {
color: white;
background-color: blue;
}

Text is white, Background is Blue

By class - .class

เหมาะกับการ set style ทั่วไป
Example
<p> Border is green </p>
.class {
border: 3px solid green
}

Border is green
