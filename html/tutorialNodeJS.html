Node JS

app.get('/your route name', (req, res) => { res.sendFile(path.join(__dirname + '/contribute/your path file')) })
คุณสามารถ copy route นี้ไปใส่ในไฟล์ index.js เพื่อเป็นหนทางเข้าสู่หน้า web page ของคุณ
route name - ส่วนนี้จะปรากฏอยู่ที่ url ตัวอย่างเช่น www.frontober.com/your-route-name
path file - เราแนะนำให้คุณสร้าง directory หรือ folder เป็นชื่อของคุณเองไว้ใน /contribute สำหรับเก็บไฟล์ HTML ของคุณ ตัวอย่างเช่น /contribute/your-name/index.html
