HTML

HTML - Hyper Text Markup Language เปรียบเสมือนโครงสร้างหลักของเว็บไซต์ ง่ายต่อการเรียนรู้

การสอนนี้ เราจะเลือก element ที่ใช้งานบ่อยๆมา และแบ่งออกเป็นกลุ่มตามการใช้งาน เพื่อนๆสามารถสามารถเลือกเอาไปใช้งานได้เลย

แบ่งกลุ่ม จัดรูปแบบ

<div> ... </div>
ใช้สำหรับแบ่งกลุ่มของแท็ก HTML ให้สะดวกต่อการเพิ่ม style และสะดวกต่อการอ่านโค้ด
<span> ... </span>
ใช้สำหรับแบ่งข้อความให้สะดวกต่อการเพิ่ม style

ข้อความ หัวข้อ

<h1> ... </h1>
<h2> ... </h2>
<h3> ... </h3>
<h4> ... </h4>
<h5> ... </h5>
<h6> ... </h6>
ข้อความที่เป็นหัวข้อ มีระดับความสำคัญตั้งแต่ h1 - h6
<p> ... </p>
ข้อความ paragraph

ปุ่ม ลิ้งค์

<a> ... </a>
ลิ้งค์ไปยังหน้าเว็บที่กำหนด
attribute
  • href - ลิ้งค์ url ที่ต้องการลิ้งค์ไปหา
<button> ... </button>
ปุ่มกดเพื่อไปยังลิ้งค์ หรือส่งข้อมูลจากฟอร์ม

รูปภาพ

<img>
แสดงรูปภาพ
attribute
  • src - path หรือ ลิ้ง url ของรูปภาพที่ต้องการแสดง
  • alt - คำประกอบรูปภาพ จะแสดงก็ต่อเมื่อไม่สามารถแสดงรูปภาพได้

ตาราง

<table> ... </table>
แสดงตาราง
<thead> ... </thead>
ส่วนหัวตาราง
<tbody> ... </tbody>
ส่วนข้อมูลในตาราง
<th> ... </th>
หัวข้อในตาราง
<tr> ... </tr>
แถวในตาราง
<td> ... </td>
คอลัมน์ในตาราง

ฟอร์มกรอกข้อมูล

<form> ... </form>
ฟอร์มจัดการการกรอกข้อมูล การส่งข้อมูล
attribute
  • action - ลิ้งค์ url ที่ต้องการส่งข้อมูลไปหลังกดปุ่มส่งฟอร์ม
  • method - restful method เช่น GET POST PUT DELETE และอื่นๆ
<input>
ส่วนกรอกข้อมูล
attribute
  • type - ประเภทของ input เช่น text checkbox radio และอื่นๆ
  • name - ชื่อของ input จะถูกกำหนดเป็นตัวแปรเก็บค่าใน input หลังส่งฟอร์ม
  • value - ค่าใน input
<label> ... </label>
ข้อความกำกับ input
